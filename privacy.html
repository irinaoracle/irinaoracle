Политика конфиденциальности

← Вернуться назад

Настоящая Политика конфиденциальности описывает порядок обработки персональных данных при посещении данного сайта.

1. Обрабатываемые данные

При загрузке сайта автоматически обрабатывается только один технический параметр:

— IP‑адрес пользователя (обрабатывается сервером для передачи данных и не используется для идентификации).

Другие персональные данные на сайте не собираются и не обрабатываются.

2. Цель обработки

Обработка IP‑адреса осуществляется исключительно для технической передачи данных и корректного отображения сайта в браузере пользователя.

3. Передача данных третьим лицам

Персональные данные не передаются третьим лицам и не используются для маркетинга или профилирования.

4. Ссылки на внешние сервисы

На сайте могут быть размещены ссылки на сторонние мессенджеры и сервисы. Переход по ссылке осуществляется по инициативе пользователя.

Обработка персональных данных при использовании таких сервисов регулируется политиками конфиденциальности соответствующих сервисов.

5. Веб‑аналитика

На сайте используется сервис GoatCounter для простой статистики посещений. Сервис не использует cookies и не собирает персональные данные. Передаваемая информация включает только технические данные браузера, посещённые страницы и анонимные события (например, нажатия на кнопки и ссылки). Вся информация обрабатывается исключительно в обезличенной форме для улучшения работы сайта.

6. Хранение данных

Сайт не сохраняет и не накапливает персональные данные пользователей.
